# SEO Setup Guide

## Built-in SEO Features

- Dynamic meta tags per page/article
- Open Graph + Twitter cards
- JSON-LD: Article, Breadcrumb, FAQ
- Canonical URLs
- `/sitemap.xml` and `/robots.txt`
- Internal linking suggestions in AI articles
- Admin SEO score (0–100) per article

## Google Search Console

1. Verify domain at [search.google.com/search-console](https://search.google.com/search-console)
2. Submit sitemap: `https://yoursite.com/sitemap.xml`
3. Monitor Core Web Vitals and indexing

## Best Practices

- Approve only high-quality AI articles (review in admin)
- Target 1200–2500 words per post
- Use descriptive slugs and meta descriptions (120–160 chars)
- Add featured images before publishing
- Build topical authority within each niche category

## EEAT Checklist

- [ ] Author bio on author pages
- [ ] Sources/references section in articles
- [ ] Affiliate disclosure page linked in footer
- [ ] Contact page with real email
- [ ] Fact-check statistics and claims
